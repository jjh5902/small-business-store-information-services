# small-business-store-information-services

## 소상공인 상권 정보 서비스
- 지역의 상권에서 동일 업종에 대한 정보 조회
- 창업시 필요한 유용한 정보

## 사용방법
- 공공데이터 포털에서 소상공인 API KEY를 발급받아 입력 후 실행
